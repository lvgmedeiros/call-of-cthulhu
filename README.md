# O Chamado de Cthulhu - RPG Narrativo

Jogo web autocontido em PT-BR inspirado no conto de domínio público de H. P. Lovecraft.

Abra `index.html` no navegador para jogar, ou use `http://localhost:4173/index.html` se o servidor local estiver rodando. As escolhas alteram Sanidade, Perigo e Pistas, levando a finais de sobrevivência, morte pela sociedade secreta ou loucura.
